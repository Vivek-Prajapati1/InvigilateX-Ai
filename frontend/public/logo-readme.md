# InvigilateX-Ai Logos

Available logo assets (SVG, scalable):

- invigilatex-ai-icon.svg — Primary app icon (shield + eye + X), used as favicon and in compact spots.
- invigilatex-ai-icon-duotone.svg — Duotone icon suitable for dark UIs.
- invigilatex-ai-mark-gradient.svg — Gradient mark with soft shadow for hero sections.
- invigilatex-ai-mark-outline.svg — Outline mark for monochrome/printing.
- invigilatex-ai-logo.svg — Full wordmark with icon (wide).
- invigilatex-ai-logo-compact.svg — Compact wordmark with icon (header/sidebar).

Usage tips:
- Prefer SVG for crisp rendering; specify width/height via CSS.
- Use the compact logo in constrained areas, full wordmark in hero/landing.
- For very small sizes (<=32px), use the icon-only variant.
